# 米埔、大生圍及甩洲—魚塘管理協議計劃 APP 使用說明書

---

## 一、應用程式簡介

本應用程式為「米埔、大生圍及甩洲—魚塘管理協議計劃」的流動應用程式，旨在協助養魚戶提交魚塘及雀鳥相片記錄，監察管理工作成效。

---

## 二、主要功能

### 2.1 啟動與登入

1. **啟動應用程式**：開啟 APP 後會顯示啟動畫面
2. **歡迎頁面**：點擊「開始使用」進入主頁
3. **登入**：如需使用提交、記錄及相簿功能，須先登入帳戶

### 2.2 首頁功能

首頁提供六個主要功能入口：

| 功能 | 說明 |
|------|------|
| **提交魚塘相片** | 拍攝/上傳魚塘水位相片，記錄降水工作階段 |
| **提交雀鳥相片** | 拍攝/上傳雀鳥使用魚塘的相片 |
| **我的魚塘記錄** | 查看已提交的魚塘相片記錄 |
| **我的雀鳥記錄** | 查看已提交的雀鳥相片記錄 |
| **魚塘相簿** | 瀏覽所有魚塘相片 |
| **雀鳥相簿** | 瀏覽所有雀鳥相片 |

---

## 三、提交相片

### 3.1 提交魚塘相片

#### 相片要求
- **拍攝階段**：降水前、基本降水後第1日、基本降水後第7日、乾塘後第1日、乾塘後第7日
- **最少3張相片**：
  - 最少1張全景相片：清楚展示整個魚塘
  - 最少2張近景相片：清楚展示魚塘水位
- **必須為日間拍攝**

#### 不合符要求的相片
- 相片模糊不清
- 晚上拍攝的相片
- 無法清楚顯示魚塘
- 同一時間拍攝多個魚塘

#### 操作步驟
1. 點擊「提交魚塘相片」
2. 選擇魚塘
3. 選擇期別（降水階段）
4. 點擊「相片要求」查看相片規範
5. 選擇拍攝新相片或從相簿選擇
6. 確認相片及位置資訊
7. 提交

### 3.2 提交雀鳥相片

#### 相片要求
- 拍攝雀鳥使用該魚塘的照片
- 相片須能夠清晰顯示雀鳥使用該魚塘

#### 不合符要求的相片
- 相片無法顯示雀鳥使用該魚塘
- 相片模糊不清

#### 備註
- 切勿過於接近雀鳥，以免驚擾雀鳥

#### 操作步驟
1. 點擊「提交雀鳥相片」
2. 選擇魚塘
3. 選擇期別（非降水/乾塘時、乾塘後、基本降水後）
4. 點擊「相片要求」查看相片規範
5. 選擇拍攝新相片或從相簿選擇
6. 確認相片及位置資訊
7. 提交

## 六、常見問題

### Q1：為什麼無法提交相片？
**A**：請確認已登入帳戶。若未登入，系統會提示您先登入。

### Q2：相片位置資訊不正確怎麼辦？
**A**：請確保已授權 APP 使用位置權限，並在拍攝時開啟 GPS 定位功能。

### Q3：可以提交多少張相片？
**A**：每次提交最多可上傳 6 張相片。魚塘相片建議至少提交 3 張（1全景 + 2近景）。

### Q4：如何查看已提交的記錄？
**A**：點擊首頁的「我的魚塘記錄」或「我的雀鳥記錄」，或使用底部導航的「我的記錄」。
